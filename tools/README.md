# 快速配置工具

本目录包含自动配置 MCP Session Saver 的脚本。

## 可用脚本

### quick-setup.sh (macOS/Linux)

自动配置脚本，支持：
- Qoder
- Cursor
- Claude Desktop
- Windsurf
- Trae
- Codebuddy
- Lingma
- Continue
- Aider

**使用方法**：
```bash
cd tools
bash quick-setup.sh
```

### quick-setup.ps1 (Windows)

Windows PowerShell 自动配置脚本，支持相同的 IDE。

**使用方法**：
```powershell
cd tools
.\quick-setup.ps1
```

## 功能特性

- ✅ 交互式选择 IDE
- ✅ 自定义存储路径
- ✅ 自动验证环境
- ✅ 一键配置所有 IDE
